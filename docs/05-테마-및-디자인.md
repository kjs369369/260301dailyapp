# 테마 및 디자인

## 다크 모드 전용

### 색상 팔레트

| 용도 | 색상 코드 | 설명 |
|------|-----------|------|
| 배경 | `#0D0D0F` | 근흑색 |
| 카드/서피스 | `#1A1A2E` | 딥 네이비 |
| 카드 강조 | `#222240` | 밝은 네이비 |
| Primary | `#00D4AA` | 틸 그린 (체크박스, 선택 상태) |
| Secondary | `#FFB74D` | 앰버 (연속 달성, 성취) |
| 텍스트 (주요) | `#EEEEEE` | 밝은 회색 |
| 텍스트 (보조) | `#9E9E9E` | 중간 회색 |
| 에러 | `#FF5252` | 빨강 |
| 성공 | `#69F0AE` | 밝은 초록 |

### 디자인 원칙
- 카드 border-radius: **16px**
- 그림자 대신 **색상 차이**로 깊이감 표현
- 기분 선택 이모지: 😢 😕 😐 🙂 😄 (1~5)

### 관련 파일
- `lib/core/theme/app_colors.dart` - 색상 상수
- `lib/core/theme/app_theme.dart` - ThemeData 정의
- `lib/core/constants/app_strings.dart` - 한국어 UI 문자열
