# 데이터베이스 설계

## 개요
- ORM: **Drift** (SQLite 래퍼, 타입 안전 쿼리 + 리액티브 스트림)
- 웹: **WebDatabase** (IndexedDB 기반, sql.js 사용)
- 모바일: **SqfliteQueryExecutor** (SQLite 파일)

## 플랫폼별 DB 연결

```
connection/
├── connection.dart    # export 'native.dart' if (dart.library.html) 'web.dart'
├── native.dart        # Android/iOS → LazyDatabase + SqfliteQueryExecutor
└── web.dart           # Web → WebDatabase('daily_app')
```

웹에서는 `sql.js` 라이브러리가 필요하며, `web/index.html`에 CDN으로 로드됨.

## 테이블 정의

### Habits (습관)
| 컬럼 | 타입 | 설명 |
|------|------|------|
| id | INTEGER (PK, auto) | |
| name | TEXT | 습관 이름 |
| emoji | TEXT | 이모지 |
| description | TEXT (nullable) | 설명 |
| sortOrder | INTEGER (default: 0) | 정렬 순서 |
| frequency | TEXT (default: 'daily') | daily / weekdays / custom |
| customDays | TEXT (nullable) | JSON 문자열 |
| isActive | BOOLEAN (default: true) | 활성 여부 |
| colorHex | TEXT (default: '0xFF00D4AA') | 색상 코드 |
| createdAt | DATETIME | 생성일 |

### HabitRecords (습관 기록)
| 컬럼 | 타입 | 설명 |
|------|------|------|
| id | INTEGER (PK, auto) | |
| habitId | INTEGER (FK) | 습관 ID |
| date | DATETIME | 날짜 |
| isCompleted | BOOLEAN (default: false) | 완료 여부 |
| note | TEXT (nullable) | 메모 |
| completedAt | DATETIME (nullable) | 완료 시각 |

- Unique 제약: (habitId, date)

### JournalEntries (일기)
| 컬럼 | 타입 | 설명 |
|------|------|------|
| id | INTEGER (PK, auto) | |
| date | DATETIME (unique) | 날짜 |
| content | TEXT | 내용 |
| title | TEXT (nullable) | 제목 |
| mood | INTEGER (nullable) | 기분 1~5 |
| createdAt | DATETIME | 생성일 |
| updatedAt | DATETIME (nullable) | 수정일 |

## 주요 쿼리 (app_database.dart)

### 습관
- `watchAllHabits()` - 활성 습관 목록 (스트림)
- `getAllHabits()` - 활성 습관 목록
- `insertHabit()` / `updateHabit()` / `deleteHabitById()` - CRUD
- `reorderHabits(orderedIds)` - 순서 변경 (batch)

### 습관 기록
- `watchRecordsForDate(date)` - 날짜별 기록 (스트림)
- `getRecordsForHabitInRange(habitId, start, end)` - 기간별 특정 습관 기록
- `getRecordsInRange(start, end)` / `watchRecordsInRange()` - 기간별 전체 기록
- `toggleHabitRecord(habitId, date)` - 완료 토글 (없으면 생성, 있으면 반전)

### 일기
- `watchJournalForDate(date)` - 날짜별 일기 (스트림)
- `watchAllJournals()` - 전체 일기 (스트림)
- `upsertJournal(date, content, title, mood)` - 생성 또는 수정
- `deleteJournal(id)` - 삭제
- `getJournalsInRange(start, end)` - 기간별 조회
