# 매일 습관 - 프로젝트 개요

## 앱 소개
매일 해야 할 습관을 체크하고 하루 일과를 기록하는 크로스플랫폼 앱

## 기술 스택

| 영역 | 기술 | 비고 |
|------|------|------|
| 프레임워크 | Flutter 3.38.7 | Android, iOS, Web |
| 상태관리 | Riverpod | flutter_riverpod ^2.6.1 |
| 로컬 DB | Drift (SQLite) | drift ^2.28.2 |
| 차트 | fl_chart | ^1.1.1 |
| 캘린더 | table_calendar | ^3.2.0 (한국어) |
| 라우팅 | GoRouter | ^17.1.0 (하단 탭 네비) |
| 디자인 | 다크 모드 전용 | 딥 네이비 + 틸 그린 |

## 앱 식별 정보

| 항목 | 값 |
|------|-----|
| 앱 이름 | 매일 습관 |
| 패키지명 | `com.kjs369.dailyhabit` |
| 버전 | 1.0.0+1 |

## 주요 기능

1. **홈 (대시보드)** - 오늘의 습관 체크리스트 + 달성률 링 + 일기 퀵카드
2. **캘린더** - 월별 달성 현황 + 날짜별 상세 (바텀시트)
3. **통계** - 주간 바 차트, 월간 라인 차트, 연속 달성일, 전체 달성률
4. **설정** - 습관 관리 (추가/수정/삭제/순서 변경)
