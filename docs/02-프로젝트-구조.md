# 프로젝트 구조

## 디렉토리 구조 (Feature-First)

```
lib/
├── main.dart                          # 앱 진입점 (ProviderScope, 한국어 로케일)
├── app.dart                           # MaterialApp.router (다크 테마)
├── core/
│   ├── constants/
│   │   └── app_strings.dart           # 한국어 UI 문자열 상수
│   ├── database/
│   │   ├── app_database.dart          # Drift DB 정의 + 모든 쿼리
│   │   ├── app_database.g.dart        # 자동 생성 코드 (build_runner)
│   │   ├── connection/
│   │   │   ├── connection.dart        # 플랫폼별 조건부 export
│   │   │   ├── native.dart            # Android/iOS: SqfliteQueryExecutor
│   │   │   └── web.dart               # Web: WebDatabase (IndexedDB)
│   │   └── tables/
│   │       ├── habits_table.dart      # 습관 테이블
│   │       ├── habit_records_table.dart # 습관 기록 테이블
│   │       └── journal_entries_table.dart # 일기 테이블
│   ├── providers/
│   │   └── database_provider.dart     # DB 인스턴스 프로바이더
│   ├── router/
│   │   └── app_router.dart            # GoRouter + StatefulShellRoute (4탭)
│   ├── theme/
│   │   ├── app_colors.dart            # 색상 상수
│   │   └── app_theme.dart             # 다크 ThemeData
│   ├── utils/
│   │   └── date_utils.dart            # 날짜 유틸리티
│   └── widgets/
│       └── app_scaffold.dart          # 공통 하단 네비게이션 바
├── features/
│   ├── home/
│   │   ├── presentation/
│   │   │   ├── home_screen.dart       # 홈 대시보드
│   │   │   └── widgets/
│   │   │       ├── today_summary_card.dart  # 달성률 링
│   │   │       ├── habit_tile.dart          # 습관 체크 타일
│   │   │       └── quick_journal_card.dart  # 일기 퀵카드
│   │   └── providers/
│   │       └── home_providers.dart
│   ├── habits/
│   │   └── presentation/
│   │       ├── habit_management_screen.dart  # 습관 관리 (순서변경)
│   │       └── widgets/
│   │           └── habit_form_dialog.dart    # 습관 추가/수정 다이얼로그
│   ├── journal/
│   │   └── presentation/
│   │       ├── journal_editor_screen.dart    # 일기 편집기
│   │       └── widgets/
│   │           └── mood_selector.dart        # 기분 선택 (1~5)
│   ├── calendar/
│   │   ├── presentation/
│   │   │   ├── calendar_screen.dart          # 월별 캘린더
│   │   │   └── widgets/
│   │   │       └── day_detail_sheet.dart     # 날짜별 상세 바텀시트
│   │   └── providers/
│   │       └── calendar_providers.dart
│   └── statistics/
│       ├── presentation/
│       │   ├── statistics_screen.dart        # 통계 메인
│       │   └── widgets/
│       │       ├── weekly_chart.dart         # 주간 바 차트
│       │       ├── monthly_chart.dart        # 월간 라인 차트
│       │       ├── streak_display.dart       # 연속 달성일
│       │       └── completion_rate_card.dart  # 전체 달성률
│       └── providers/
│           └── statistics_providers.dart
```

## 네비게이션 (하단 탭 4개)

```
[홈] → HomeScreen
[캘린더] → CalendarScreen
[통계] → StatisticsScreen
[설정] → HabitManagementScreen
```
